1.2.5 #1.2.5 Float

The float classes are defined as .sy-float-{value}. The value can be none, left, right or inherit.

Example
Markup 
<div class="sy-w-3 sy-h-3 sy-float-left sy-blue-500"></div>
<div class="sy-w-3 sy-h-3 sy-float-right sy-red-500"></div>

1.2.2 #1.2.2 Padding

Padding classes are defined as .sy-p{direction}-{size}. The size ranges from 0 (smallest) up to 5 (biggest).

Example

sy-px-0 sy-pb-3

sy-px-5

sy-px-0 sy-pt-3

Markup 
<p class="sy-px-0 sy-pb-3 sy-blue-500 sy-text-mono">sy-px-0 sy-pb-3</p>
<p class="sy-px-5 sy-blue-500 sy-text-mono">sy-px-5</p>
<p class="sy-px-0 sy-pt-3 sy-blue-500 sy-text-mono">sy-px-0 sy-pt-3</p>

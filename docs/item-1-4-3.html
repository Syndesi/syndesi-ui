1.4.3 #1.4.3 Column

In contrast to the row can the column be filled with content and is responsible for its own width. The class is defined as .sy-col-{width}, whereas the width ranges from 1 (1/12 = 8.3%) to 12 (12/12 = 100%). When no width is defined (e.g. .sy-col), the column will use as much space as possible and can therefore be used for divisions with the same width, e.g. 5 columns with equal width in a single row.
